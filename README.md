<div align="center">

| Attribute | Detail |
|:---|:---|
| **Stack** | React.js · Node.js · Python · YOLOv8 · FastAPI |
| **Scale** | Multi-class animal detection · Real-time inference on uploaded media |
| **Performance** | 80%+ detection accuracy on diverse wildlife image datasets |
| **Architecture** | Python inference microservice decoupled from the Node.js application layer |
| **Impact** | Accessible ecological intelligence tool with a clean web interface |

The frontend renders bounding box overlays on returned images with species labels and confidence values.

</details>
